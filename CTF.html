Welcome to my walkthrough of Over The Wire Bandit. Over The Wire is a CTF type wargame where the user will use the linux terminal to find flags.

**Passwords found in this walkthrough may be outdated. Over The Wire occasionally changes passwords and levels!**

Intro:

The goal of this level is for you to log into the server using SSH. To do this we must first open a linux terminal and type in the following command:
ssh bandit0@bandit.labs.overthewire.org -p 2220.
This securely connects to the server address using SSH and the -p is the port! and bandit0 is the level

The password is: bandit0

Make sure to exit!

Level 0:

First enter the correct ssh adress and enter the password obtained in the last level when prompted too. The password for the this level is stored in a file called readme located in the home directory, to find this we must ls and cat the output, readme

Make sure to exit!

Level 1:

First enter the correct ssh adress and enter the password obtained in the last level when prompted too. The password for the this level is stored in a file called - located in the home directory, to find this we must ls and cat the output, ./-

Make sure to exit!

Level 2:

First enter the correct ssh adress and enter the password obtained in the last level when prompted too. The password for the this level is stored in a file called spaces in this filename located in the home directory, to find this we must ls and cat the output with \ after each word
